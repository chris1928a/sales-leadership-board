Teil 2: Training Methoden

Closed-Loop Coaching
Vom Call zur Verhaltensänderung

Call Scoring (6 Dimensionen)

  • Opener → Hook-Rotation, Pattern-Interrupt, Gatekeeper-Bypass
  • Pitch → Value Prop, Problem-first, ROI-Argumentation
  • Einwandbehandlung → LARC Framework, Reframing, Poke-the-Bear
  • Closing → 1-10 Close, Hypothetical, Summary, Trial, Assumptive
  • Gesprächstiefe → SPIN Selling, Mirroring, 3-Sek-Pause
  • Rapport → Labeling, Active Listening, Tonalität

Warum diese Methoden?

  • LARC (Listen-Acknowledge-Respond-Confirm): Strukturierte Einwandbehandlung statt Improvisation
  • SPIN Selling: Situation-Problem-Implication-Need. Prospects verkaufen sich selbst.
  • Pattern-Interrupt: Erste 7 Sekunden entscheiden. Standardöffner = Auflegen.
  • 3-Sek-Pause: Nach jeder Frage. Prospect füllt die Stille mit echten Informationen.

Ergebnis (14-Tage Rolling)

System identifiziert schwächste Dimension pro Rep und mappt automatisch auf das passende Training-Modul. Keine generischen Tipps.

Takeaways

3 Dinge die ihr
heute mitnehmen könnt

1. Reporting ist nicht Coaching

Gong zeigt euch WAS passiert. Ein Coaching-System sagt jedem Rep WAS ER ÜBEN SOLL. Das sind zwei verschiedene Probleme. Die meisten Tools lösen nur das erste.

2. Call-Recordings sind eine Goldmine

Nicht für "Deal Intelligence". Sondern für ICP-Validierung, Pitch-Optimierung und Einwand-Bibliotheken. Eure Prospects sagen euch GENAU was sie brauchen. Ihr müsstet nur systematisch zuhören.

3. Ihr braucht kein 17.000 EUR Tool

Python + CRM API + Claude + Deepgram. 15 EUR/Monat. In 3 Tagen gebaut. Der Code ist auf GitHub. Copy-paste-ready für Close.com und HubSpot.
